# Vessel Flute Drawing Brief

## Required Views

- Hero perspective or CAD render showing final sculptural form.
- Top/hand view showing hole placement, grip, and mouthpiece direction.
- Side view showing fipple axis, body depth, and split line.
- Section through chamber showing internal volume, wall target, and port necks.
- Fipple detail with windway, labium, window, and edge thickness.
- Mold breakdown showing parting lines, registration keys, and pull directions.

## Critical Dimensions

| Dimension | Why it matters |
| --- | --- |
| Chamber volume | Primary pitch variable |
| Open port/hole areas | Primary pitch variable |
| Effective neck length | Model sensitivity and tuning correction |
| Windway gap | Response and breath pressure |
| Labium angle/edge | Tone onset and clarity |
| Wall thickness | Weight, tap tone, drying, and durability |
| Mold split | Manufacturability |
| Shrinkage scale factor | Final fired dimensions |

## Drawing Notes

- Mark all dimensions as target fired dimensions or master dimensions.
- Add a build ID and CAD revision to every drawing.
- Use `TBD` for any fipple value not yet validated by a speaking prototype.
- Include a validation table on the drawing once measurements exist.
