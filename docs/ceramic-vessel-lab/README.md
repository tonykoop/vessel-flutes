# Ceramic Vessel Lab Addendum - Vessel Flutes

This folder preserves the brainstorm/design packet that was developed in the shared `instrument-maker` workspace before the material was copied into the dedicated `vessel-flutes` repo.

Use the repo-root files for the current vessel-flute lab packet:

- `../../design.md`
- `../../concept-matrix.csv`
- `../../experiments.csv`
- `../../validation.csv`
- `../../mold-workflow.md`
- `../../drawing-brief.md`
- `../../wolfram-starter.wl`

Use this addendum for the copied shared-lab context:

- `design.md` - experimental vessel-flute family brief.
- `concept-matrix.csv` - VF-01 through VF-07 concept set.
- `experiments.csv` - physics and process questions to test.
- `validation.csv` - prototype measurement log.
- `mold-workflow.md` - 3D-printed master to plaster mold workflow.
- `drawing-brief.md` - drawing and CAD view requirements.
- `wolfram-starter.wl` - compact Helmholtz/water-fill model starter.
- `ceramic-vessel-instrument-lab.md` - cross-instrument ceramic vessel roadmap.

Nothing in this addendum overwrites the root-level packet; it is here as provenance and research context.
